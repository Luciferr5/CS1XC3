1. ls - look at the surroundings 
2. cd LOCATION - move to a new location 
3. cd - takes us to home 
4. less ITEM - interact with things 
5. add LOCKERNAME - gives us extra spells 
6. help - shows a list of available spells
7. man SPELLNAME - gives details on how to use specific spells  
8. mv - moves things to another place 
9. mkdir NAME - makes a new place 
10. pwd - show file patch of working parent directory
11. add - adds items 
12. touch ITEM - makes an item 
13. cp - copies one file content to another 
14. rm - removes certain files 
15. sudo - allows actions on any specific item 
16. grep - searches file for certain item or line
